the consulate — the governing body

“All agents finished” is not “the work is correct.”

Most AI workflows end when the model stops talking. Macrothought's orchestration layer is a deterministic pipeline that turns intent into verified, gated releases — a completed agent ≠ a completed requirement ≠ a shippable release.

intent→ requirements→ context→ routing→ execution→ verification→ adversarial audit→ repair→ release gate→ metrics

deterministic spine

No model in the loop. The requirement ledger, verification harness and release gates are plain state and plain code. Completion is provable, not asserted.

local intelligence

Cheap and idempotent. Local models classify, compile context and propose simple fixes — propose-only, never silently deciding. Runs on the desk.

frontier intelligence

Where judgment matters. Frontier agents take design, hard repair and adversarial audit. Everything they claim is re-verified by the spine.

what a governed brain is for

Built on files. Not about files.

The same pattern — one readable vault, explicit confidence, verified releases — fits anywhere knowledge decays faster than people can maintain it.

A 40-PERSON COMPANY
Ops knowledge lives in one vault instead of six wikis nobody trusts. When a process changes, the note is updated, not appended — and the release gate means the runbook that ships is the runbook that was verified. onboarding drops from weeks to days.
A TEACHER
A curriculum as a living department: lessons wikilinked to standards, past cohorts' stumbling points tagged [uncertain] until re-tested. Recall is gap-aware — the system says what's stale before you teach it again. prep from the map, not from memory.
A CONTENT CREATOR
Research, scripts, footage and renders mapped as one tree per series. The pipeline treats an episode like a release: drafts don't ship until the checks pass — sources cited, assets present, cut verified. the archive becomes the studio.
A HOUSEHOLD'S FINANCES
Decisions recorded with their reasoning and confidence at the time — then re-checked on a weekly verify-or-decay cadence instead of trusted forever. No advice, no black box: an audit trail you can read in plain text. every claim dated, every claim checkable.

the philosophy

Memory you can audit.

01

Markdown first. Databases, graph stores and vector memories were tried — and parked. The institutional memory is text a human can read, diff and version.

02

Confidence is explicit. Every claim is tagged [verified] [inferred] or [uncertain] — dated, and re-checked weekly.

03

Gap-aware recall. Answers say what's stale and what's missing. A memory system that implies completeness is lying to you.

04

No queue without a drain. Every automation that accumulates work has a routine that consumes it. Machinery that only grows is machinery that rots.
